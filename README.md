# AI Powered Regulatory Compliance Checker for Contracts

This project is an AI-powered tool that helps check contracts for regulatory compliance.

## Features

- Automated contract analysis
- Regulatory compliance checking
- AI-powered insights using Groq LLM

## Setup

1. Clone the repository
```bash
git clone https://github.com/soumyakanera104-source/soumyakanera.git
```

2. Create and activate virtual environment
```bash
python -m venv myvenv
# On Windows
.\myvenv\Scripts\Activate.ps1
# On Unix/MacOS
source myvenv/bin/activate
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Set up environment variables
Create a `.env` file and add your Groq API key:
```
GROQ_API_KEY=your_api_key_here
```

## Usage

Run the application:
```bash
python app.py
```
